/// ABOUT_

I'm Jacques — a self-taught web developer from South Africa with a passion for clean, atmospheric design and brutalist aesthetics. I build websites that feel like something: moody restaurant pages, high-energy gym sites, boutique hotel experiences. Each project is an exercise in atmosphere first, code second.

I work with HTML, CSS, and JavaScript — the core of the web. I focus on responsive, accessible design that works on any device, and I deploy everything through modern CI/CD pipelines so updates are seamless.

My approach is simple: I don't use templates. Every site I build starts from a blank file and a vision.
